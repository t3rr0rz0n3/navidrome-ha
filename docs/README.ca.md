# Integració Navidrome per a Home Assistant

Integració personalitzada per a Home Assistant que permet connectar amb el teu servidor Navidrome mitjançant l’API Subsonic.

Proporciona informació en temps real sobre la reproducció, estadístiques de la biblioteca i estat del servidor.

## 📦 Instal·lació (HACS)

1. Obre HACS
2. Ves a **Integracions**
3. Menú ⋮ → **Repositoris personalitzats**
4. Afegeix: https://github.com/t3rr0rz0n3/navidrome-ha
5. Tipus: **Integration**
6. Instal·la
7. Reinicia Home Assistant

---

## ⚙️ Configuració

1. Configuració → Dispositius i serveis
2. Afegeix integració → Navidrome
3. Introdueix:
   - URL del servidor
   - Usuari
   - Contrasenya

---

## ✨ Característiques

- Reproductor amb informació de reproducció
- Botó per escanejar la biblioteca
- Estadístiques:
  - Total de cançons
  - Total d’artistes
  - Total de gèneres
- Sensors de reproducció:
  - Cançó actual
  - Artista actual
  - Àlbum actual
- Últim escaneig (temps relatiu)
- Multi-idioma
- Configuració des de la UI
