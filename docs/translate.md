# 🌍 Translation Guide

Want to help translate this integration into your language? Awesome! 🎉

This integration uses Home Assistant's built-in translation system.

---

## 📁 Translation files

All translations are located in: custom_components/navidrome/translations/

Each language has its own file:

+ en.json
+ es.json
+ ca.json
+ ...
  
---

## ➕ Adding a new language

1. Copy an existing file (for example `en.json`)
2. Rename it using your language code:

fr.json → French
de.json → German
it.json → Italian

👉 Use ISO 639-1 language codes

---

## ✏️ Translate the content

Translate only the **values**, never the keys.

---

## 🚀 Submitting your translation

+ Fork the repository
+ Add your translation file
+ Open a Pull Request
